# CampusPulse — University Event Management Website

Assignment No. 01 — Web Technologies
University of Engineering & Technology, Peshawar — Summer 2026

Built with **HTML5 and CSS3**

## Folder structure

```
campuspulse/
├── index.html        Home page
├── events.html        Events (8 event cards, CSS Grid)
├── schedule.html       Schedule (15+ row table + timeline)
├── speakers.html       Speakers (6 profiles)
├── register.html       Registration form (design only)
├── contact.html        Contact page (address, map, form, FAQ)
├── css/
│   └── style.css        Single external stylesheet
├── assets/
│   ├── download.jpeg          Site logo
│   └── map.svg            Contact page map illustration
└── README.md
```

## Design system

- **Theme:** Purple / indigo with a ceremonial gold accent — a professional
  palette for a university events brand.
- **Typography:** Fraunces (display serif) + Work Sans (body) + IBM Plex Mono
  (dates, times, labels) — loaded from Google Fonts in `style.css`.
- **Signature element:** a "boarding pass / ticket stub" card component
  (dashed perforation, punch-hole notches, ribbon badges) reused across the
  hero, events, and speakers pages to tie the visual language to the
  subject — event admission passes.
